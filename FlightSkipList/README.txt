Purpose: 
Flight database that is implemented as a skip list
and has the following features: 

Functional features:
  - Inserting flight data
  - Deleting flight data 
  - Suggests a flight with the departure date 3 days before and after requested 
    departure date if it cannot find any flights on the specified date 
  - Finds flights that have the same origin and destination cities and same date 
    as search entered, with departure times within the timeDifference part of the search 

*Note: no GUI created
